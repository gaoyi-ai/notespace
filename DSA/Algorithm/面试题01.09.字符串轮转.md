# 题目分析

- 这道题还挺有意思的

    首先，旋转字符串的特点是，某些字符从后面跑到了前面，这就好像由一队变两队，队尾变成了队头

# Solution

```java
public static boolean isRevolving(String s1, String s2) {
    if (s1 == null || s2 == null) return false;
    if (s1.length() != s2.length()) return false;
    // 这里还可以考虑使用KMP算法
    return (s1 + s1).contains(s2);
}
```
